# 动态时间显示程序

## 项目结构

本项目是一个使用32位汇编语言编写的Windows GUI应用程序，主要包含以下文件：

```
动态时间/
├── GUI.ASM     # 主汇编源代码文件
├── gui.res     # 资源文件
├── guiApp.app  # 应用程序打包文件
└── README.md   # 项目说明文档
```

## 程序功能

这是一个简单但完整的Windows GUI应用程序，主要功能是：

1. **动态显示系统时间**：实时显示当前系统的小时、分钟和秒
2. **自动更新**：每秒自动更新一次时间显示
3. **用户友好界面**：简洁的窗口界面，清晰的时间显示

## 技术实现

程序使用以下技术和API实现功能：

1. **Windows API调用**：使用Windows核心API创建窗口和控件
2. **消息循环机制**：通过Windows消息循环处理用户交互和系统事件
3. **定时器功能**：使用SetTimer API实现每秒更新时间
4. **字符串格式化**：使用wsprintf函数格式化时间字符串
5. **静态文本控件**：使用STATIC控件显示时间信息

## 教学亮点

本项目作为汇编语言教学示例，具有以下亮点：

1. **完整的Windows GUI程序结构**：展示了从窗口创建到消息处理的完整流程
2. **详细的中文注释**：代码中包含大量中文注释，解释每个API调用和关键步骤
3. **模块化设计**：清晰的代码结构，分离的窗口创建、消息处理和时间更新逻辑
4. **实用的API演示**：展示了如何使用Windows API创建控件、设置定时器、获取系统时间等实用功能
5. **适合初学者**：代码简洁明了，注释详细，适合汇编语言和Windows编程初学者学习

## 代码结构说明

### 1. 头部定义

包含汇编器指令、Windows头文件和库文件的引用，以及函数声明。

### 2. 数据段

定义了窗口类名、窗口标题、时间格式字符串等常量数据。

### 3. 未初始化数据段

定义了应用程序实例句柄、控件句柄、定时器ID等变量。

### 4. 代码段

包含程序的主要逻辑：
- **START**：程序入口点
- **WinMain**：应用程序主函数，负责窗口创建和消息循环
- **WndProc**：窗口过程函数，处理各种Windows消息

## 编译和运行

### 编译

使用MASM汇编器编译：

```bash
ml /c /coff GUI.ASM
link /subsystem:windows GUI.obj
```

### 运行

直接双击GUI.exe文件即可运行程序。
